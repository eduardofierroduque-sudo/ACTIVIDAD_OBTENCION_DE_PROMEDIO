Hola hola. Este es un pequeño script de Python diseñado para facilitar el cálculo de promedios de cualquier curso. La idea es que sea interactivo, permitiéndote ingresar tus notas una a una y obtener el resultado final al instante. 
1. Preparación y ContextoPrimero, el programa necesita saber de qué estamos hablando. Por eso, lo primero que hace es preguntarte el nombre del curso y cuántas notas vas a promediar. Esto nos permite dimensionar el trabajo del computador.
2. El Acumulador (El "Cerebro")Creamos una variable llamada suma_total que empieza en 0. Imagínala como una caja vacía donde iremos depositando cada nota que ingreses para no olvidarla.3. El Ciclo de Repetición (Bucle for)En lugar de escribir código para cada nota manualmente,7
usamos un ciclo. Si dijiste que tienes 5 notas, el programa:Te pide la nota.La guarda en la "caja" (suma_total).Repite el proceso hasta llegar a la quinta nota.4. Matemáticas SimplesUna vez que el ciclo termina y tenemos todas las notas sumadas,
